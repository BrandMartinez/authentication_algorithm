the executable (.exe) is on the route /aplicacion/bin/debug/aplicacion.exe
The passwords file is on the route /aplicacion/bin/debug/passwords.txt


The application uses a variation of the HMAC-SHA-256 algorithm

The application saves 3 codes along with the username:
The first is the number of iterations.
The second is the salt.
The third is the resulting hash code.
